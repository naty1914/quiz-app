Get Started

How It Works

Step 1: Choose a Quiz

Select from a variety of quizzes across different topics to test your knowledge.

Step 2: Answer Questions

Complete the quiz by answering multiple-choice questions within the time limit.

Step 3: Submit & Get Results

Submit your answers and immediately view your results to see your performance.

Step 4: View Detailed Feedback

Review correct, incorrect, and unanswered questions to improve your knowledge.

Key Features

User Authentication

Secure login and registration for user access to quiz results.

Dynamic Quiz Selection

Users can select quizzes like "Tech" or "General" and add questions.

REST API for Quizzes

REST API for managing quiz questions with full CRUD support.

Results Display

Results Display: Instant feedback with score and total questions.
